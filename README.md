zzst
====

## URLs

    ## https://st.suckless.org/
    ## https://st.suckless.org/patches/
    ## https://dl.suckless.org/st/st-0.9.2.tar.gz

## Build dependence

    $ sudo apt install make gcc g++
    $ sudo apt install libx11-dev libxft-dev libxinerama-dev libgd-dev

## Build st

    $ cd src
    $ make
    $ sudo make install

## Init st

    $ ./init.sh
    $ ./init.sh all
