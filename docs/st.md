st

## Fetch Source

```
$ git clone https://git.suckless.org/st
```

## Keyboard Shortcuts

Action      | Key Combination
---         | ---
Copy        | `ctrl` + `shift` + `c`
Paste       | `ctrl` + `shift` + `v`
Zoom In     | `ctrl` + `shift` + `PageUp`
Zoom Out    | `ctrl` + `shift` + `PageDown`
Reset Zoom  | `ctrl` + `shift` + `Home`


## Build errors

```
term.line[n] -> TLINE(n)
term.line[n][lastpos].mode -> TLINE(n)[lastpos].mode
```
